pyOptimizer is a performance-focused utility for Windows 10 and 11. It automates system optimization by applying registry-level tweaks, managing power configurations, and performing system maintenance tasks.

## Disclaimer
This software modifies Windows Registry values. It is recommended to create a System Restore Point before execution. The author is not responsible for any system instability resulting from the use of this tool.
